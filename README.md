Python's Rock Scissor Paper game

1.딕셔너리로 가위,바위,보 확인: 가위=0, 바위=1, 보=2

2.컴퓨터는 랜덤한 값을 가지고 플레이어는 내고 싶은 카드를 선택

3.미리 정해진 딕셔너리로 플레이어가 제출한 카드에 대응하는 숫자로 치환해서 승패를 비교 ("컴퓨터 값-플레이어 값 = -1 or 2" 라면 플레이어의 승리)

4.score 리스트에 점수 기록 & score_str로 누구의 점수인지 미리 리스트 제작 => 게임 종료 후 두 리스트를 딕셔너리로 합해서 출력

5."END" 입력시 게임이 종료되고 scoreboards 출력
